# 📌 Mini Project: Performance Trade-off Analysis of NLP Models

## 📖 About the Project
This project focuses on comparing the performance of different NLP models on multiple datasets.  
The main aim is to understand the trade-off between model performance, complexity, and efficiency.

---

## 📂 Datasets Used
- VAST dataset  
- COVID-19 dataset  
- P-Stance dataset  
- SemEval Task A dataset  
- SemEval Task B dataset  

---

## 🤖 Models Used

### 🔹 Deep Learning Models
- LSTM  
- GRU  
- MiniLSTM  
- MiniGRU  

### 🔹 Transformer Models
- BERT  
- BART  

---

## 📁 Project Structure

### 🟢 VAST, COVID-19, P-Stance
- Deep learning models (LSTM, GRU, MiniLSTM, MiniGRU) are grouped together in one file  
- Transformer models (BERT, BART) are implemented in separate files  

### 🟢 SemEval Task A & Task B
- All models (Deep Learning + Transformer) are implemented in a **single file for each task**  
  - One file for SemEval Task A  
  - One file for SemEval Task B  

---

## ⚙️ What This Project Does
- Trains multiple models on different datasets  
- Compares their performance  
- Helps understand:
  - Accuracy differences  
  - Model efficiency  
  - Trade-offs between simple and complex models  

---

## 📊 Key Idea
This project compares:
- Traditional models → LSTM, GRU  
- Lightweight models → MiniLSTM, MiniGRU  
- Advanced models → BERT, BART  

This helps in choosing the right model for different real-world problems.

---

## 🚀 How to Run
1. Open the `.ipynb` files in Google Colab or Jupyter Notebook  
2. Run all cells step by step  
3. Observe results and performance  

---

## 🎯 Conclusion
This project provides a clear comparison of different NLP models across multiple datasets and highlights the performance trade-offs between them.
